Título: Pet & Style Loja Virtual (empresa fictícia)

Descrição: Página inicial para uma loja fictícia chamada Pet & Style, especializada em produtos para pets com estilo. Foram utilizadas classes do Tailwind CSS, criando uma página responsiva e visualmente agradável, utilizando estratégias de mobile-first.

Tecnologias utilizadas: HTML5, Tailwind

Pré-requisitos: Navegador atualizado (projeto foi testado e funciona no Google Chrome Versão 142.0.7444.163 64 bits - Caso utilize este navegador, recomenda-se tal versão ou superior).

Instalação:

Passos para clonar e rodar: git clone https://github.com/tayanibritto/petstyle-loja-virtual.git / cd petstyle-loja-virtual / abrir index.html no navegador 
Como usar: Para testar a responsividade da página, você pode utilizar o DevTools do Navegador e escolher os vários tipos de dispositivos.
